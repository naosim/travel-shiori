使用方法

基本的な記述ルール

  • HH:mm イベント名 - 時刻付きイベント
  • - | 移動手段 - インターバルラベル（前のイベントと次のイベント間の説明）
  • - メモ - イベントの詳細（インデント必須）
  • - 🏨 宿泊先（宿泊） - 時刻なしイベント

JavaScriptでの使用

const markdown = \`10:00 駅出発
10:30 ホテル到着\`;

const html = DangoTimeline.render(markdown);
document.getElementById('timeline').innerHTML = html;
